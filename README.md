# local-aria2-webui

本地化的 aria2 webui

感谢原作者

https://github.com/ziahamza/webui-aria2

安装

安装 nodejs

https://nodejs.org/

```shell
apt install nodejs
```

安装 yarn

https://yarnpkg.com/

```shell
npm install -g yarn
```

安装 live-server

https://www.npmjs.com/package/live-server

```shell
yarn global add live-server

```

启动

windows

```shell
start.cmd
```

linux

```shell
start.sh
```

使用浏览器打开网址

http://127.0.0.1:8080
